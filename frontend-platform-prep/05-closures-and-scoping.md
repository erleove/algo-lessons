# Closures And Scoping

Цель: понимать области видимости, замыкания и типовые ловушки `var` в циклах.

## Что нужно уметь

- Объяснить лексическое окружение (lexical environment) и почему “функция помнит внешние переменные”.
- Отличать:
  - `var` — function scope + hoisting (инициализация `undefined`)
  - `let/const` — block scope + TDZ
- Понимать проблему:
  - `for (var i=0; i<n; i++) setTimeout(() => console.log(i))` печатает `n` много раз.

## Частые способы исправить `var`-цикл

- заменить `var` на `let`;
- замкнуть значение через IIFE: `(function(j){ ... })(i)`;
- вынести в функцию с параметром.
