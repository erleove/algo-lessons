# Node Task Templates (Platform)

Здесь лежат JavaScript-шаблоны задач по platform-темам.

## Как работать

1. Открой файл из `core` или `intermediate`.
2. Найди функцию/класс с `TODO`.
3. Реализуй решение.
4. Запусти файл через Node:

```bash
node "/Users/lerm/projects/lessons/platform-algo-prep/node-tasks/core/01-pipe.js"
```

Если все тесты пройдены, увидишь `PASS` по каждому кейсу.

## Структура

- `_lib/testRunner.js` — простой раннер тестов.
- `core` — основные platform-задачи.
- `intermediate` — задачи повышенной сложности.
