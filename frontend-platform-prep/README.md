# Platform Interview Prep (JavaScript)

Этот набор материалов помогает прокачать platform-навыки для собеседований и повседневной работы с JS runtime: async, event loop, замыкания, `this`, композиция функций.

## Как использовать

1. Начни с `01-pipe-and-composition.md`.
2. Параллельно решай задачи из `node-tasks/core`.
3. Для сверки с эталоном используй папку `answers`.

## Порядок прохождения

1. Pipe / композиция / sync+async
2. Promises / async-await / Promise.* методы
3. Event loop / microtasks vs macrotasks
4. EventEmitter (подписки, отписки, once)
5. Closures / scoping / var vs let
6. this / call, apply, bind

## Цель по времени

- Небольшая задача: 15–25 минут.
- Объяснение механики (event loop / this / closures): уверенно, без подсказок и с примерами edge cases.
