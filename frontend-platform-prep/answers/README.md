# Answers

Здесь лежат эталонные решения отдельно от шаблонов задач.

## Зачем это нужно

- можно сначала решить задачу самому в `node-tasks`;
- потом сравнить с эталонной реализацией;
- шаблоны с `TODO` при этом остаются нетронутыми.

## Как использовать

Открывай нужный файл:

- `core-answers.js`

При желании можно запускать через Node:

```bash
node "/Users/lerm/projects/lessons/platform-algo-prep/answers/core-answers.js"
```
