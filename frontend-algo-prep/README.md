# Frontend Algorithm Interview Prep

Этот набор материалов помогает подготовиться к алгоритмическому собеседованию frontend-разработчику на JavaScript/TypeScript.

## Как использовать

1. Начни с `01-core-patterns.md`.
2. После уверенного решения базовых задач переходи к `02-intermediate-topics.md`.
3. Для mock-интервью и устного объяснения используй `03-interview-delivery.md`.
4. Для недельного трекинга используй `04-practice-tracker.md`.
5. Для маршрута в стиле Blind 75 используй `05-blind-75-route.md`.
6. Для практики в Node используй папку `node-tasks`.
7. Для сверки с эталоном используй папку `answers`.

## Порядок подготовки

1. Arrays, strings, hash map, set.
2. Two pointers, sliding window, prefix sum.
3. Stack, queue, binary search, intervals.
4. Linked list.
5. Tree DFS/BFS.
6. Graph basics.
7. Dynamic programming basics.

## Цель по времени

- Easy-задачи: 10-15 минут.
- Medium-задачи знакомого паттерна: 20-30 минут.
- Объяснение сложности и edge cases: без подсказок.
